# Changelog

## 2.0.0 - 2026-07-17

- 新增共享布局主题引擎，支持原始、微信式、飞书式和 QQ 2007 复古式四种 Codex 工作台布局。
- 新增隔离的顶部工具栏、右侧任务信息栏和底部状态栏；窄窗口自动隐藏右栏。
- 顶部快捷入口可转发到 Codex 原有的新任务、文件、终端、变更、浏览器和设置功能。
- macOS 与 Windows 启动器新增布局选择器，并复用同一个 JavaScript 布局资源。
- 新增真实 Chrome 注入测试，覆盖主题启用、切换和恢复原始布局。

## 1.1.0 - 2026-07-17

- 新增 Windows 11 / x64 原生 WPF 启动器。
- 自动识别 Microsoft Store / MSIX 安装的 `OpenAI.Codex`，无需手动选择程序路径。
- Windows 版内置 CDP 注入逻辑，使用单文件自包含构建，不要求安装 Node.js 或 .NET。
- 新增 Windows Runner 自动构建、产物上传和 Release 附件流程。

## 1.0.0 - 2026-07-17

- 首次公开版本。
- 支持背景图片、颜色、字体及图片效果配置。
- 支持自动启动 Codex、实时应用和新窗口自动注入。
- 支持一键关闭皮肤并恢复原始外观。
