# Changelog

## 2.1.0 - 2026-07-17

- macOS 启动器新增“显示实时额度血条”开关，默认开启，在原始布局和三种工作台布局下都能独立显示。
- 通过 Codex App Server 的 `account/rateLimits/read` 获取初始额度，并监听 `account/rateLimits/updated` 实时合并稀疏更新。
- 血条按当前最紧张的额度窗口显示剩余百分比，支持绿色、黄色、红色分级；悬停展示额度桶、窗口、重置倒计时和 Credits 信息。
- App Server 断开后保留最后一次快照并自动重连；30 秒轮询作为事件通知的兜底。
- 新增额度客户端单元测试、稀疏更新测试和真实 Chrome 血条注入测试。
- Codex 主窗口侧边栏新增项目和任务实时筛选框，可按名称搜索并在“全部 / 执行中”之间切换。
- 执行中数量会跟随任务状态、任务新增和页面重绘自动更新；清除筛选后完整恢复原生列表。
- macOS 与 Windows 共用同一筛选模块，桌面宠物及辅助浮窗继续保持不注入。
- 新增真实 Chrome 侧边栏筛选测试，覆盖项目匹配、任务匹配、执行状态变化、空结果和恢复列表。

## 2.0.1 - 2026-07-17

- 皮肤和工作台布局现在只注入 Codex 主工作区窗口。
- 排除桌面宠物的 `avatar-overlay` 页面及其 composition surface，避免背景、透明度和布局影响宠物显示。
- 新增多窗口 CDP 测试，验证主窗口正常换肤且模拟宠物窗口保持原样。

## 2.0.0 - 2026-07-17

- 新增共享布局主题引擎，支持原始、微信式、飞书式和 QQ 2007 复古式四种 Codex 工作台布局。
- 新增隔离的顶部工具栏、右侧任务信息栏和底部状态栏；窄窗口自动隐藏右栏。
- 顶部快捷入口可转发到 Codex 原有的新任务、文件、终端、变更、浏览器和设置功能。
- macOS 与 Windows 启动器新增布局选择器，并复用同一个 JavaScript 布局资源。
- 新增真实 Chrome 注入测试，覆盖主题启用、切换和恢复原始布局。

## 1.1.0 - 2026-07-17

- 新增 Windows 11 / x64 原生 WPF 启动器。
- 自动识别 Microsoft Store / MSIX 安装的 `OpenAI.Codex`，无需手动选择程序路径。
- Windows 版内置 CDP 注入逻辑，使用单文件自包含构建，不要求安装 Node.js 或 .NET。
- 新增 Windows Runner 自动构建、产物上传和 Release 附件流程。

## 1.0.0 - 2026-07-17

- 首次公开版本。
- 支持背景图片、颜色、字体及图片效果配置。
- 支持自动启动 Codex、实时应用和新窗口自动注入。
- 支持一键关闭皮肤并恢复原始外观。
